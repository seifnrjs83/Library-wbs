# 도서관리 프로그램 WBS (2주 계획)

## 프로젝트 목표

Vue + Spring + OracleDB를 활용한 도서관리 CRUD 프로그램 제작
mvc패턴 이용하고, 백엔드 mybatis 이용해서 구현.

기술 스택:

* Vue
* Spring Boot
* OracleDB
* MyBatis
* VSCode
* Eclipse
