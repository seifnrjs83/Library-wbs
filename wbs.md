# 도서관리 프로그램 WBS

## 프로젝트 목표

Vue + Spring + OracleDB를 활용한 도서관리 CRUD 프로그램 제작
mvc 패턴 이용하고, 백엔드는 myBatis 이용해서 구현예정.

기술 스택:

* Vue
* Spring Boot
* OracleDB
* VSCode
* Eclipse
